== Description ==

AI Engine Theme is designed for modern websites that require clean layouts, responsive design, and flexible customization options. The theme includes customizable homepage sections, multiple page templates, typography controls, color settings, and integration support for popular plugins. It is built with optimized code structure and supports responsive viewing across desktops, tablets, and mobile devices. The layout system allows easy arrangement of content blocks, service sections, portfolios, blogs, and contact pages. It also includes navigation customization, banner management, social media integration, and widget-ready areas for additional functionality. The theme supports fast-loading pages and compatibility with major browsers for consistent performance. Built-in customization settings help manage branding elements such as fonts, colors, logos, and header styles without requiring extensive coding changes. AI Engine Theme also supports SEO-friendly structure and accessibility-focused design practices for improved usability and content organization across different types of websites and digital platforms.

== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the ai-engine theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away
